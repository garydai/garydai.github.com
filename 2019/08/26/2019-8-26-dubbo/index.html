    服务导入

    消费者也要注册

    org.apache.dubbo.registry.integration.RegistryProtocol#refer

    1. 从注册中心找到所有服务提供者的地址
    2. 按路由策略进行过滤
    3. 包装所有服务提供者的地址在一个对象内
    4. 注册监听器以监听路由策略的变化、服务提供者地址的变化、服务调用相关信化
