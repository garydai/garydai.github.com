Java log

log4j

jul

jcl

apache.common.log，遍历一个class string数组，class.forname，如果有class则生成该class实例，目前有log4j、jul两种类

slf4j

simple logging facade for java（门面）

通过绑定器绑定不同的log实现

提供一个桥接器，将其他日志框架桥接到slf4j

spring log
