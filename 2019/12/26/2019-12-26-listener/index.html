观察者模式

jdk的实现

被观察者通知观察者

被观察者，Observable

添加观察者，Observable.addObserver(Observer)

通知观察者，Observable.notifyObservers(Object) { observers.update() }

观察者，Observer，根据被观察者传来的事件来做处理，逻辑处理在观察者

通知观察者的是被观察者

spring的实现

事件发布及监听

观察者（监听器），监听事件，不再是监听被观察者

逻辑处理在观察者

事件，event

被观察者被管理器替代，由管理器发布事件，spring根据利用事件的范型触发对应的监听器

通知观察者的是被管理器
