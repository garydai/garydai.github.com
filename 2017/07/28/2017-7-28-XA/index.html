BASE 理论

基本可用（Basically Available）：指分布式系统在出现故障时，允许损失部分的可用性来保证核心可用。

软状态（SoftState）：指允许分布式系统存在中间状态，该中间状态不会影响到系统的整体可用性。

最终一致性（EventualConsistency）：指分布式系统中的所有副本数据经过一定时间后，最终能够达到一致的状态。
